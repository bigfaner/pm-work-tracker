# 1.5: SubItem ChangeStatus Update & AvailableTransitions

## Description

Update SubItem's ChangeStatus to use the new `pkg/status` transitions, remove deprecated status logic (已延期/待验收/挂起), update the return type to `SubItemChangeResult`, and add AvailableTransitions method + handler.
