# 2.1: Admin service — ResetPassword and SoftDeleteUser

## Description

Implement the two new service methods on `AdminService` following the existing handler→service→repo pattern.

## Tasks

### 1. ResetPassword

```go
func (s *adminService) ResetPassword(ctx context.Context, targetBizKey int64, newPassword string) (*dto.ResetPasswordResp, error)
```

- Find user by `bizKey` using `FindByBizKey` (with `NotDeleted` scope)
- If not found → return `ErrUserNotFound`
- Hash `newPassword` with bcrypt (cost 10, matching existing pattern)
- Update user's `Password` field
- Return `ResetPasswordResp` with `BizKey`, `Username`, `DisplayName`

### 2. SoftDeleteUser

```go
func (s *adminService) SoftDeleteUser(ctx context.Context, callerID uint, targetBizKey int64) error
```

- Find user by `bizKey` using `FindByBizKey` (with `NotDeleted` scope)
- If not found → return `ErrUserNotFound`
- If `callerID == user.ID` → return `ErrCannotDeleteSelf`
- Call `repo.SoftDelete(ctx, user)`
- Return nil
