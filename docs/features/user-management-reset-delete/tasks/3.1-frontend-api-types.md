# 3.1: Frontend API module and TypeScript types

## Description

Add TypeScript request/response types and API functions for reset password and delete user endpoints. These are the frontend data layer consumed by UI components.

## Tasks

### 1. TypeScript types

Add to `frontend/src/types/index.ts` (or the appropriate types file):

```ts
interface ResetPasswordReq {
  newPassword: string;
}

interface ResetPasswordResp {
  bizKey: string;
  username: string;
  displayName: string;
}
```

### 2. API functions

Add to `frontend/src/api/admin.ts`:

```ts
function resetPasswordApi(userId: string, req: ResetPasswordReq): Promise<ResetPasswordResp>
function deleteUserApi(userId: string): Promise<void>
```

- `resetPasswordApi`: `client.put(`/api/v1/admin/users/${userId}/password`, req)`
- `deleteUserApi`: `client.delete(`/api/v1/admin/users/${userId}`)`

### 3. Unit tests

- Mock `client` and verify both functions call the correct endpoints with correct payloads
- Follow existing test patterns in `frontend/src/api/`
