# 3.2: Extract shared VO & service helpers

## Description

Create shared `TeamVO`, `UserVO` structs replacing untyped `gin.H` responses, and `RecordStatusChange` helper replacing 5 duplicate status-history recording call sites.

- `backend/internal/vo/team_vo.go` (new) — TeamVO + NewTeamVO
- `backend/internal/vo/user_vo.go` (new) — UserVO + NewUserVO
- `backend/internal/service/status_history_helper.go` (new) — RecordStatusChange

- `backend/internal/handler/auth_handler.go` — userToDTO → NewUserVO

- `backend/internal/service/main_item_service.go` — 3 status history call sites
