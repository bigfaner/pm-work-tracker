# T-test-2: Generate e2e Test Scripts

## Description

Call `/gen-test-scripts` skill to generate executable TypeScript e2e test scripts from test cases.

Generated scripts use:
- UI tests: Playwright Locator API (semantic locators)
- API tests: Node.js built-in `fetch`
- CLI tests: `child_process.execSync`
- Test framework: `node:test` + `node:assert`

## Reference Files

- `testing/test-cases.md` — Test case document (generated by T-test-1)
- `docs/sitemap/sitemap.json` — Page element locators (required for UI tests)
- `tests/e2e/config.yaml` — Test environment config

## Acceptance Criteria

- [ ] `testing/scripts/package.json` created
- [ ] `testing/scripts/helpers.ts` created
- [ ] At least one spec file generated (ui.spec.ts / api.spec.ts / cli.spec.ts)
- [ ] Each test() includes traceability comment `// Traceability: TC-NNN → {PRD Source}`
