# 4.2: Merge constructors & VO conversion

## Description

Merge `NewViewService` / `NewViewServiceWithUserRepo` into a single constructor with optional `userRepo` parameter. Merge single/batch VO conversion functions in `item_pool_handler` and `progress_handler` into unified functions.

## Reference Files

- `backend/internal/service/view_service.go` — two constructors (Item 20)
- `backend/internal/handler/item_pool_handler.go` — itemPoolToVO + itemPoolsToVOs (Item 21)
- `backend/internal/handler/progress_handler.go` — progressRecordToVO + progressRecordsToVOs (Item 21)
- `backend/internal/router/router.go` — constructor call sites
- `docs/features/schema-alignment-cleanup/design/tech-design.md` — Items 20, 21
