# 4.3: Item Pool Page

## Description

Rebuild the item pool page (视觉重做, interaction unchanged).

**Page layout (`/item-pool`):**
- Filter bar: status filter (待分配/已分配/已拒绝), search by title/ID
- Item cards with colored left border (待分配=blue, 已分配=gray, 已拒绝=red)
- Infinite scroll loading
- Action buttons: 转换为主事项, 转换为子事项, 拒绝
- Dialog forms for each action (pre-fill from pool item data)

## Reference Files

- `docs/features/improve-ui/ui/prototype/item-pool.html` — design reference
- `frontend/src/pages/ItemPoolPage.tsx` — current implementation

## Acceptance Criteria

- [ ] Page matches prototype visual design
- [ ] Left border color indicates pool item status
- [ ] Filter and search work correctly
- [ ] Infinite scroll loads more items
- [ ] Convert to main item dialog works (priority, assignee, dates)
- [ ] Convert to sub item dialog requires parent item selection
- [ ] Reject dialog requires reason text
- [ ] No antd imports

- Reuse `ItemFilters` component from shared components.
