# 4.2: Main Item + Sub Item Detail Pages

## Description

Rebuild two detail pages with visual redesign (interaction logic unchanged):

**Main item detail page (`/items/:mainItemId`):**
- Breadcrumb navigation: 事项 > [主事项标题]
- Header: ID badge, title, priority badge, status badge, action buttons (edit, archive)
- Info grid: assignee (with avatar), start date, expected end date, actual end date
- Circular progress indicator (Radix Progress or custom SVG)
- Collapsible sections: 成果汇总 (green), 卡点汇总 (red)
- Sub items table with inline status change dropdown
- "创建子事项" button → dialog

**Sub item detail page (`/items/:mainItemId/sub/:subItemId`):**
- Breadcrumb: 事项 > [主事项标题] > [子事项标题]
- Header: ID, parent item link, priority, status, assignee (with avatar), expected end date
- Linear progress bar
- "追加进度" button → dialog (percentage, achievements, blockers)
- Progress timeline (reverse chronological): date, percentage, achievements, blockers
