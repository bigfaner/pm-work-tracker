# 5.5: Team Detail Page (Independent Route)

## Description

Create the new team detail page as an independent route, split off from the old admin page's inline view. Supports full member management.

**Page layout (`/teams/:teamId`):**
- Breadcrumb: 团队管理 > [团队名]
- Team info card: name, PM (with avatar), member count, created date, description
- Member list table: name (with avatar), role (PM/member badge), joined date, actions
  - PM row: no action buttons
  - Member row: "设为 PM" button, "移除" button
- "添加成员" button → dialog (search/select user)
- "解散团队" button → ConfirmDialog with team name input verification
- Danger zone section with disband button
