# 4.5: Team Management Page

## Description

Rebuild the team management (team list) page. Visual redo only — interaction logic unchanged.

**Page layout (`/teams`):**
- Team cards in a grid/list layout
- Each card: team name (clickable → team detail), description, PM (with avatar), member count, created date
- "创建团队" button → dialog
