# 5.4: User Management Page (Independent Route)

## Description

Create the new independent user management page, split off from the old admin page. This is a SuperAdmin-only page with full user CRUD.

**Page layout (`/users`):**
- User table: name (with avatar), username (monospace), email, teams (badge list), canCreateTeam (switch), status badge, actions (edit, toggle status)
- Search: username/name fuzzy search input
- Filter: canCreateTeam dropdown (all/yes/no)
- Pagination: default 10/page
- "创建用户" button → dialog
- Edit user dialog (pre-fill current values)
- Toggle status dialog with confirmation

**New interaction — Create user success:**
- After successful creation, show a dialog with the auto-generated initial password
- Message: "请妥善保管，关闭后无法再次查看"
- User confirms → dialog closes
