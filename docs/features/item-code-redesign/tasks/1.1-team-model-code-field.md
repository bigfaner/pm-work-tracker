# 1.1: Team 模型新增 Code 字段

## Description

在 `model.Team` 结构体中新增 `Code` 字段（varchar(6)，全局唯一索引）。同步更新 `dto.CreateTeamReq` 和 `dto.UpdateTeamReq`（Code 不可修改，UpdateTeamReq 不加此字段）。

## Reference Files

- `backend/internal/model/team.go`
- `backend/internal/dto/team_dto.go`
- `docs/features/item-code-redesign/design/tech-design.md` — Data Models: model.Team

## Acceptance Criteria

- [ ] `model.Team` 包含 `Code string` 字段，GORM tag: `type:varchar(6);not null;uniqueIndex:idx_teams_code`，JSON tag: `json:"code"`
- [ ] `dto.CreateTeamReq` 新增 `Code string` 字段，binding tag: `required,min=2,max=6,alpha`
- [ ] `dto.UpdateTeamReq` 不包含 Code 字段（Code 不可修改）
- [ ] `go build ./...` 无编译错误

## Implementation Notes

`alpha` 是 go-playground/validator 内置 tag，校验仅含字母字符（a-z, A-Z）。
