## Implementation Notes

SQLite 不支持 `DROP COLUMN`（3.35.0 以下），回滚脚本中注明需重建表。`DEFAULT ''` 允许迁移后现有行有空 code，数据迁移程序（任务 4.1）负责填充实际值。
