# 5.1: 前端：TeamManagementPage 新增 Code 输入框

## Description

在 `TeamManagementPage` 的创建团队对话框中新增 Code 输入框，含前端格式校验（2~6 位字母）和后端唯一性错误展示。团队列表新增 Code 列。更新 `frontend/src/api/teams.ts` 的请求/响应类型。

## Reference Files

- `frontend/src/pages/TeamManagementPage.tsx`
- `frontend/src/api/teams.ts`
- `docs/features/item-code-redesign/prd/prd-ui-functions.md` — UI Function 1, 2

## Acceptance Criteria

- [ ] 创建团队对话框包含 Code 输入框，placeholder 为 "如 FEAT、CORE"
- [ ] 输入框失焦时校验格式（`/^[A-Za-z]{2,6}$/`），不合法显示 "编码须为 2~6 位英文字母"
- [ ] 后端返回 `TEAM_CODE_DUPLICATE` 错误时，输入框下方显示 "该编码已被使用"
- [ ] 团队列表每行显示 Code 列
- [ ] `CreateTeamReq` 类型新增 `code: string` 字段
- [ ] `Team` 类型新增 `code: string` 字段
- [ ] 前端测试通过：`npm test`

## Implementation Notes

Code 输入框使用现有 `Input` 组件。错误展示复用现有表单错误展示模式（参考 name 字段的错误展示）。
