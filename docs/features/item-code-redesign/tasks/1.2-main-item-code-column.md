# 1.2: MainItem.Code 列扩展 varchar(12)

## Description

将 `model.MainItem` 的 `Code` 字段 GORM tag 从 `varchar(10)` 扩展到 `varchar(12)`，以容纳 `{TEAM_CODE}-NNNNN` 格式（最长 6+1+5=12 字符）。

## Reference Files

- `backend/internal/model/main_item.go`
- `docs/features/item-code-redesign/design/tech-design.md` — Data Models: model.MainItem

## Acceptance Criteria

- [ ] `model.MainItem.Code` 的 GORM tag 为 `type:varchar(12);not null;uniqueIndex:idx_main_item_team_code`
- [ ] `go build ./...` 无编译错误

## Implementation Notes

联合唯一索引 `idx_main_item_team_code`（team_id + code）已在 migration 006 中创建，GORM tag 中的 `uniqueIndex` 名称需与之一致，避免重复创建索引。
