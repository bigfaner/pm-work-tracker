# 3.1: TeamService.CreateTeam() 传入 Code 字段

## Description

更新 `service/team_service.go` 的 `CreateTeam()` 方法，将 `req.Code` 写入 `model.Team.Code`。更新 `TeamHandler.Create()` 的 `teamToDTO()` 函数新增 `code` 字段。

## Reference Files

- `backend/internal/service/team_service.go:47`（CreateTeam 方法）
- `backend/internal/handler/team_handler.go:209`（teamToDTO 函数）
- `docs/features/item-code-redesign/design/tech-design.md` — Interfaces: Handler: teamToDTO

## Acceptance Criteria

- [ ] `CreateTeam()` 将 `req.Code` 赋值给 `team.Code`
- [ ] `teamToDTO()` 返回值包含 `"code": team.Code`
- [ ] `TeamListResp` 和 `TeamDetailResp` DTO 新增 `Code string` 字段（json:"code"）
- [ ] `ListTeams()` 和 `GetTeamDetail()` 响应中包含 `code` 字段
- [ ] 单元测试：`TestCreateTeam` 验证 Code 字段被正确写入
