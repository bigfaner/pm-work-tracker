# 3.2: SubItemService.Create() 调用 NextSubCode()

## Description

在 `service/sub_item_service.go` 的 `Create()` 方法中，调用 `SubItemRepo.NextSubCode()` 生成编码并写入 `SubItem.Code`。

## Reference Files

- `backend/internal/service/sub_item_service.go`（Create 方法）
- `backend/internal/repository/sub_item_repo.go`（接口）
- `docs/features/item-code-redesign/design/tech-design.md` — Architecture: Component Diagram

## Acceptance Criteria

- [ ] `SubItemService.Create()` 在创建前调用 `s.subItemRepo.NextSubCode(ctx, req.MainItemID)`
- [ ] 生成的 code 写入 `SubItem.Code`
- [ ] NextSubCode 失败时，Create 返回错误，不创建子事项
- [ ] 单元测试：mock `NextSubCode` 返回 `FEAT-00001-01`，验证 SubItem.Code 被正确赋值
