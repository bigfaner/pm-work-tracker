# 1.3: SubItem 模型新增 Code 字段

## Description

在 `model.SubItem` 结构体中新增 `Code` 字段（varchar(15)，per-main-item 唯一索引）。同步更新 `SubItemRepo` 接口新增 `NextSubCode()` 方法签名。

## Reference Files

- `backend/internal/model/sub_item.go`
- `backend/internal/repository/sub_item_repo.go`
- `docs/features/item-code-redesign/design/tech-design.md` — Data Models: model.SubItem；Interfaces: SubItemRepo

## Acceptance Criteria

- [ ] `model.SubItem` 包含 `Code string` 字段，GORM tag: `type:varchar(15);not null`，JSON tag: `json:"code"`
- [ ] `sub_items` 表有 per-main-item 唯一索引（`main_item_id, code` 联合唯一）
- [ ] `repository.SubItemRepo` 接口新增 `NextSubCode(ctx context.Context, mainItemID uint) (string, error)`
- [ ] `go build ./...` 无编译错误（GORM 实现暂可返回空字符串占位）

## Implementation Notes

GORM 联合唯一索引写法：`uniqueIndex:"idx_sub_item_main_code"` 配合 `MainItemID` 字段上的同名 tag，或在迁移 SQL 中手动创建。建议在迁移 SQL 中创建，避免 GORM AutoMigrate 行为不确定。
