# 10.3: 表格视图 Page

## Description
Implement the table view page with an antd Table, sort/filter controls, and a CSV export button.

## Reference Files
- `ui/ui-design.md` — Navigation (表格视图 nav item)
- `api-handbook.md` — GET /views/table and GET /views/table/export
- `prd/prd-spec.md` — Section 5.6 (table view fields, sort/filter rules, CSV export)

## Acceptance Criteria
- Filter bar: Type Select (主事项/子事项/全部), Priority Select, Status Select, Assignee Select
- antd `Table` columns: 编号, 标题, 类型 Tag, 优先级 Tag, 负责人, 状态 Tag, 完成度 Progress, 预期完成时间, 实际完成时间
- Sortable columns: 编号, 优先级, 完成度, 预期完成时间, 实际完成时间 (click column header to toggle asc/desc)
- Pagination: pageSize=50, shows total count
- "导出 CSV" button calls `GET /views/table/export` with current filters; triggers browser file download
- CSV export button shows loading state during download; shows `message.error` on NO_DATA
- Default sort: priority DESC, expectedEndDate ASC
- Loading: antd Table `loading` prop; empty state via Table `locale.emptyText`

## Implementation Notes
- Use antd Table `onChange` callback to capture sort state and re-fetch with updated `sortBy`/`sortOrder` params
- CSV download: use `URL.createObjectURL(new Blob([csvBytes], { type: 'text/csv' }))` and programmatically click an `<a>` tag
- `useQuery` key includes all filter + sort + pagination params so cache is per-combination
- Type Tag: "主事项" (blue) / "子事项" (default gray)
