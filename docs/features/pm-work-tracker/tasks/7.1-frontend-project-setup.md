# 7.1: Frontend Project Setup

## Description
Bootstrap the frontend project with Vite + React 18 + TypeScript and install all required dependencies. Configure path aliases, Vite proxy, and base TypeScript/ESLint settings.

## Reference Files
- `design/tech-design.md` — Dependencies section (frontend package list)
- `design/tech-design.md` — Component Diagram (frontend directory structure)

## Acceptance Criteria
- `frontend/vite.config.ts` configures `@` alias to `src/`, and proxy `/api` → `http://localhost:8080`
- `frontend/tsconfig.json` has `strict: true`, `paths` matching the `@` alias
- All packages installed: `antd`, `@tanstack/react-query`, `zustand`, `axios`, `react-router-dom@6`, `dayjs`, `frappe-gantt`
- Dev dependencies: `vitest`, `@testing-library/react`, `@testing-library/user-event`, `msw`, `@types/react`, `@types/node`
- `frontend/src/types/index.ts` exports shared TypeScript interfaces for `User`, `Team`, `MainItem`, `SubItem`, `ProgressRecord`, `ItemPool` matching the API data contracts
- `npm run dev` starts the dev server without errors
- `npm run build` produces a dist bundle without TypeScript errors

## Implementation Notes
- Antd v5 (not v4) — uses CSS-in-JS, no need to import antd CSS manually
- Configure `dayjs` locale to `zh-cn` globally in `main.tsx`
- Set up `QueryClient` with default `staleTime: 30000` and `retry: 1`
