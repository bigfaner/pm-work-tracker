# 1.5: MainItemService + SubItemService + ItemPoolService: teamID uint → teamBizKey int64

## Description

Update three service interfaces and implementations:

**MainItemService** (`main_item_service.go`): all methods with `teamID uint` originating from URL context → `teamBizKey int64`

**SubItemService** (`sub_item_service.go`): all methods with `teamID uint` originating from URL context → `teamBizKey int64`

**ItemPoolService** (`item_pool_service.go`): all methods with `teamID uint` originating from URL context → `teamBizKey int64`; fix `TeamKey: int64(teamID)` → `TeamKey: teamBizKey` in `Submit`
