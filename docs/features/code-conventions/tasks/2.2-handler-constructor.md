# 2.2: Handler Constructor Validation

## Description

Replace per-method nil-service checks with constructor-time panic validation across all 6 handlers. Remove 31 method-level nil checks.

**Current pattern (31 occurrences)**:
```go
func (h *XxxHandler) Method(c *gin.Context) {
    if h.svc == nil {
        c.JSON(http.StatusNotImplemented, gin.H{"error": "service not configured"})
        return
    }
    // ...
}
```

**Target pattern**:
```go
func NewXxxHandler(svc XxxService) *XxxHandler {
    if svc == nil {
        panic("xxx_handler: xxxService must not be nil")
    }
    return &XxxHandler{svc: svc}
}
```

**Handlers to update**: MainItemHandler, SubItemHandler, ItemPoolHandler, TeamHandler, RoleHandler, AdminHandler

**Panic format**: `panic("<handler_snake_case>: <service_interface_name> must not be nil")`
