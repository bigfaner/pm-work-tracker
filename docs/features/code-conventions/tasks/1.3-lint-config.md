# 1.3: Configure Lint Rules

## Description

Add lint rules to enforce the conventions defined in task 1.2.

**Backend** — update `backend/.golangci.yml`:
- Enable `tagliatelle` with `json: camel` rule
- Enable `dupl` with threshold 80

**Frontend** — create/update `frontend/eslint.config.js`:
- `no-restricted-syntax` to block hardcoded Tailwind colors (emerald-*, red-*, amber-*, slate-*)
