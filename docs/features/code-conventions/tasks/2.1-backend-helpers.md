# 2.1: Create Backend Shared Helpers

## Description

Create 3 shared helper functions with unit tests, following the interface definitions in api-handbook.md.

**1. `pkg/errors.MapNotFound`** (`backend/internal/pkg/errors/`):
- Generic function: `func MapNotFound(err error, domainErr *AppError) error`
- Maps `gorm.ErrRecordNotFound` and `ErrNotFound` to domain error
- Returns other errors unchanged
- Replaces: 5 domain-specific mapXxxNotFound functions

**2. `dto.ApplyPaginationDefaults`** (`backend/internal/dto/`):
- `func ApplyPaginationDefaults(page, pageSize int) (offset, page, pageSize int)`
- Defaults: page=1, pageSize=20
- Computes offset = (page-1) * pageSize
- Replaces: 6 inline pagination blocks

**3. `pkg/dates.ParseDate`** (new file `backend/internal/pkg/dates/dates.go`):
- `func ParseDate(s string) (time.Time, error)`
- Uses `time.Parse("2006-01-02", s)`
- Exports `DateFormat = "2006-01-02"` constant
- Replaces: 11 inline time.Parse calls
