# 2.2: Fix Migration Layer (P3/P4)

## Description

Fix the remaining two incompatibility points in the migration layer:

- **P3**: `rebuildTeamMembersTable` uses `INTEGER PRIMARY KEY AUTOINCREMENT` DDL which is SQLite-only. Add `isMySQL` branch with MySQL-compatible DDL (`BIGINT UNSIGNED NOT NULL AUTO_INCREMENT` + `PRIMARY KEY(id)`).
- **P4**: `HasColumn` delegates to `columnExists` internal function (which already has `isMySQL` branching via `information_schema`). Verify delegation is correct and add MySQL path if not yet done.

These changes use the existing `isMySQL()` helper — no dependency on the `Dialect` interface.
