# 2.3b: MainItem service + handler 对齐

## Description

以 `pmw_main_items` 表为单位，完成 service 和 handler 层对齐。依赖 2.3a（model + repo 已对齐）。

**Service（`backend/internal/service/main_item_service.go`）**：
- `Create()` 赋值 `BizKey: snowflake.Generate()`，捕获 MySQL 1062 返回 `ErrDuplicateBizKey`
- 新增 `GetByBizKey(ctx context.Context, bizKey int64) (*model.MainItem, error)`
- 更新 `MainItemService` 接口定义（新增 `GetByBizKey`）
- 更新 service 内部字段引用（`Status` → `ItemStatus`，`TeamID` → `TeamKey` 等）

**Handler（`backend/internal/handler/main_item_handler.go`）**：
- `:itemId` 路径参数解析改为 `strconv.ParseInt`（bizKey int64）
- GET/PUT 调用 `svc.GetByBizKey(bizKey)`
- DELETE：先 `svc.GetByBizKey(bizKey)` 取 record，再 `svc.Delete(record.ID)`（内部 ID 不对外暴露）
