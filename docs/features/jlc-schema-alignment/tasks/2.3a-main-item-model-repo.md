# 2.3a: MainItem model + repo 对齐

## Description

以 `pmw_main_items` 表为单位，完成 model 和 repo 层对齐。

**Model（`backend/internal/model/main_item.go`）**：
- 嵌入新 `BaseModel`
- `Status` → `ItemStatus`（`json:"itemStatus"`）
- `Description` → `ItemDesc`（`json:"itemDesc"`）
- `StartDate` → `PlanStartDate`（`json:"planStartDate"`）
- FK 字段：`TeamID uint` → `TeamKey int64`，`ProposerID uint` → `ProposerKey int64`，`AssigneeID *uint` → `AssigneeKey *int64`
- `TableName()` → `pmw_main_items`

**Repo 实现（`backend/internal/repository/gorm/main_item_repo.go`）**：
- 所有 `Find`/`First`/`List`/`Count` 加 `.Scopes(NotDeleted)`
- 实现 `FindByBizKey(ctx, bizKey int64)`
- 更新 `List`、`ListByTeamAndStatus`、`ListNonArchivedByTeam`、`filter_helpers.go` 中的字段引用（`Status` → `ItemStatus`，`TeamID` → `TeamKey` 等）
- 更新 `rewrite_codes.go` 中如有字段引用
