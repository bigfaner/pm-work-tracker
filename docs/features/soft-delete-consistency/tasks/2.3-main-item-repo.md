# 2.3: Add NotDeleted to main_item_repo methods

## Description

Add `.Scopes(NotDeleted)` to all query methods in `main_item_repo.go`.

## Reference Files

- `backend/internal/repository/gorm/main_item_repo.go` — 6 methods to modify
- `docs/features/soft-delete-consistency/design/tech-design.md` — Module 4 table

No direct user story mapping. Defensive coverage for future delete feature.

## Implementation Notes

All methods use single-table queries on `pmw_main_items`. Each adds `.Scopes(NotDeleted)` after `.WithContext(ctx)`. The `FindByBizKeys` method builds a raw query — add the scope before `.Where("biz_key IN ?", bizKeys)`.
