# 2.2: Add NotDeleted to team_repo methods

## Description

Add `.Scopes(NotDeleted)` or `.Scopes(NotDeletedTable(...))` to team_repo query methods. After TeamMember is unified to soft-delete, FindMember and CountMembers also need NotDeleted on `pmw_team_members`.

## Reference Files

- `backend/internal/repository/gorm/team_repo.go` — 8 methods to modify
- `backend/internal/repository/gorm/scopes.go` — NotDeleted, NotDeletedTable
- `docs/features/soft-delete-consistency/design/tech-design.md` — Module 3 table, Decision 2

## Acceptance Criteria

- [ ] `List` adds `.Scopes(NotDeleted)` — excludes soft-deleted teams
- [ ] `ListFiltered` adds `.Scopes(NotDeleted)` — excludes from search
- [ ] `FindByBizKey` adds `.Scopes(NotDeleted)` — returns error for soft-deleted team
- [ ] `ListMembers` adds `.Scopes(NotDeletedTable("pmw_users"))` — excludes deleted users from member list
- [ ] `FindMember` adds `.Scopes(NotDeleted)` on pmw_team_members — excludes soft-deleted members
- [ ] `CountMembers` adds `.Scopes(NotDeleted)` on pmw_team_members — counts only active members
- [ ] `FindPMMembers` adds `.Scopes(NotDeletedTable("pmw_users"))` — excludes deleted users
- [ ] `FindTeamsByUserIDs` adds `.Scopes(NotDeletedTable("pmw_teams"))` — excludes deleted teams
- [ ] New test: soft-deleted team not found by FindByBizKey
- [ ] New test: soft-deleted team not in List
- [ ] New test: soft-deleted member not found by FindMember
- [ ] New test: CountMembers excludes soft-deleted members

## User Stories

- Story 4 AC1: HasPermission excludes deleted member (team_repo FindMember supports this by not finding deleted members)
- PRD Coverage Map: TeamMember FindMember excludes deleted
- PRD Coverage Map: TeamMember CountMembers excludes deleted

## Implementation Notes

`FindMember` and `CountMembers` query `pmw_team_members` which embeds BaseModel. After the TeamMember soft-delete unification (task 3.1), these methods must filter by `NotDeleted`. For `ListMembers`, `FindPMMembers` which join `pmw_users`, add `NotDeletedTable("pmw_users")`. For `FindTeamsByUserIDs` which joins `pmw_teams`, add `NotDeletedTable("pmw_teams")`.
