# 1.5: 新增路由中间件权限码测试（7 个用例）

## Description

在 `backend/internal/handler/router_test.go` 中新增 7 个测试用例，验证新权限码绑定在无权限时返回 403：

- `role:read`：`GET /admin/roles`、`GET /admin/roles/:id`、`GET /admin/permissions`
- `role:create`：`POST /admin/roles`
- `role:update`：`PUT /admin/roles/:id`
- `role:delete`：`DELETE /admin/roles/:id`
- `user:list`：`GET /admin/users`
- `user:read`（新语义）：`GET /admin/users/:userId`
- `user:assign_role`：`POST /admin/users`

（role:read 覆盖 3 条路由，可合并为 1 个测试函数；共 7 个独立测试用例）

## Implementation Notes
