# 1.1: 更新权限码注册表（permissions/codes.go）

## Description

更新 `backend/internal/pkg/permissions/codes.go` 中的权限码注册表：
- `user` 资源：将 3 个码更新为 4 个码（`user:read` → `user:list`，新增 `user:read`（新语义），`user:manage_role` → `user:assign_role`）
- 新增 `role` 资源：4 个码（`role:read`、`role:create`、`role:update`、`role:delete`）

- `backend/internal/pkg/permissions/codes.go` — 权限码注册表
