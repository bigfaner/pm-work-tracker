# 3.1: Implement ServeStatic and ServeSPA handlers

## Description

新建 `backend/internal/handler/static.go`，实现两个 Gin handler：
- `ServeStatic(fsys fs.FS) gin.HandlerFunc`：服务 `dist/assets/*` 静态文件，带长期缓存头
- `ServeSPA(fsys fs.FS) gin.HandlerFunc`：始终返回 `dist/index.html`，带 no-cache 头

## Implementation Notes

参数类型用 `fs.FS`（而非 `embed.FS`），便于测试时传入 `fstest.MapFS`。

`ServeStatic` 从 Gin 路径参数 `filepath` 读取文件路径，拼接 `dist/assets/` 前缀后在 fsys 中查找。

MIME 类型通过文件扩展名推断（`mime.TypeByExtension`），同时设置 `X-Content-Type-Options: nosniff`。

```go
func ServeStatic(fsys fs.FS) gin.HandlerFunc {
    return func(c *gin.Context) {
        filePath := "dist/assets/" + c.Param("filepath")
        f, err := fsys.Open(filePath)
        if err != nil {
            c.JSON(http.StatusNotFound, gin.H{"error": "not found"})
            return
        }
        defer f.Close()
        // set cache headers, detect MIME, write response
        c.Header("Cache-Control", "max-age=31536000, immutable")
        c.Header("X-Content-Type-Options", "nosniff")
        // ...
    }
}
```
