# 7.1: Create scripts/build.sh

## Description

新建 `scripts/build.sh`，实现一键构建前后端单体二进制的完整流程。

## Reference Files

- `scripts/build.sh` — 新建

## Acceptance Criteria

- [ ] 接受 `dev` 或 `prod` 参数，其他参数打印用法并退出码 1
- [ ] `dev` 环境校验当前分支为 `dev`，`prod` 校验为 `main`，不匹配则退出码 1 并打印错误
- [ ] 执行 `npm ci && npm run build`（在 frontend 目录）
- [ ] 将 `frontend/dist/` 复制到 `backend/web/dist/`
- [ ] 执行 `go build -o ../bin/pm-work-tracker-{env} ./cmd/server`（在 backend 目录）
- [ ] 任意步骤失败立即停止（`set -e`），非零退出码
- [ ] 成功后打印 `Build complete: bin/pm-work-tracker-{env}`
- [ ] 脚本有执行权限（`chmod +x`）

## Implementation Notes

```bash
#!/usr/bin/env bash
set -euo pipefail

ENV="${1:-}"
SCRIPT_DIR="$(cd "$(dirname "${BASH_SOURCE[0]}")" && pwd)"
ROOT_DIR="$(dirname "$SCRIPT_DIR")"

# Validate ENV
if [[ "$ENV" != "dev" && "$ENV" != "prod" ]]; then
    echo "Usage: $0 <dev|prod>" >&2
    exit 1
fi

# Validate branch
EXPECTED_BRANCH="$([[ "$ENV" == "prod" ]] && echo "main" || echo "dev")"
CURRENT_BRANCH="$(git -C "$ROOT_DIR" rev-parse --abbrev-ref HEAD)"
if [[ "$CURRENT_BRANCH" != "$EXPECTED_BRANCH" ]]; then
    echo "Error: $ENV build requires branch '$EXPECTED_BRANCH', current branch is '$CURRENT_BRANCH'" >&2
    exit 1
fi

# Build frontend
cd "$ROOT_DIR/frontend"
npm ci
npm run build

# Copy to backend embed dir
rm -rf "$ROOT_DIR/backend/web/dist"
cp -r "$ROOT_DIR/frontend/dist" "$ROOT_DIR/backend/web/dist"

# Build backend
mkdir -p "$ROOT_DIR/bin"
cd "$ROOT_DIR/backend"
go build -o "../bin/pm-work-tracker-$ENV" ./cmd/server

echo "Build complete: bin/pm-work-tracker-$ENV"
```
