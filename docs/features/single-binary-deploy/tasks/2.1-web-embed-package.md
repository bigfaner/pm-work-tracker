## Implementation Notes

```go
package web

import (
    "embed"
    "fmt"
    "io/fs"
)

//go:embed dist
var FS embed.FS

func ValidateAssets(fsys embed.FS) error {
    _, err := fsys.Open("dist/index.html")
    if err != nil {
        return fmt.Errorf("embedded assets missing dist/index.html — did the build script succeed?")
    }
    return nil
}
```

测试使用 `testing/fstest.MapFS` 转换为 `embed.FS` 的替代方案：直接用 `fs.FS` 接口。
注意：`ValidateAssets` 参数类型可改为 `fs.FS` 以便测试时传入 `fstest.MapFS`。
