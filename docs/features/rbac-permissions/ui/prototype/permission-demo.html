实现原理

usePermission(code) — Hook 从 auth store 读取权限数据：

  • isSuperadmin = true → 直接返回 true
  • 在团队上下文 → 检查 teamPermissions[currentTeamId] 是否包含 code
  • 非团队上下文 → 检查所有团队权限合集是否包含 code

<PermissionGuard code="..."> — 包裹组件：

  • 有权限 → 渲染子组件
  • 无权限 → 不渲染（不占空间，非置灰）

权限刷新时机：

  • 登录后请求 GET /api/me/permissions
  • 路由切换时检查缓存（5 分钟过期）
  • 团队切换时立即刷新
  • 角色变更后 5 分钟轮询检测
