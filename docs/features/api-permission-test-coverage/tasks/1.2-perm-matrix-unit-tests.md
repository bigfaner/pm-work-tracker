# 1.2: 12 TestPermMatrix_* unit test functions

## Description

## User Stories

**Story 1 — 权限差异端点的单元测试**

- Given 一个 handler 测试，通过 `c.Set("permCodes", []string{"main_item:archive"})` 注入权限
- When 调用 `POST /archive` 端点
- Then 响应码为 200

- Given 一个 handler 测试，注入空 `permCodes`（`[]string{}`）
- When 调用同一端点
- Then 响应码为 403

## Implementation Notes

- 每个测试函数使用 `buildPermTestRouter`（1.1 实现）构建路由
- mock service 的方法在 "has permission" case 中被调用，需要设置 `On(...).Return(...)` 或直接返回零值
- 对于需要路径参数的端点（如 `/:itemId/archive`），路由注册为 `/test/:itemId`，请求路径为 `/test/1`
- 不测试 handler 业务逻辑，只断言 status code
