# 2.1: F1: Sidebar 菜单重排与分组

## Description

调整 `Sidebar.tsx` 中菜单项的排列顺序，将业务功能和管理功能分为两组，中间加分隔线。

## Reference Files

- `frontend/src/components/layout/Sidebar.tsx`
- `docs/features/ui-ux-unification/design/tech-design.md` — F1 节

## Acceptance Criteria

- [ ] 业务组顺序：事项清单 → 待办事项 → 每周进展 → 整体进度 → 周报导出
- [ ] 管理组顺序：团队管理 → 用户管理 → 角色管理
- [ ] 管理组第一项（团队管理）上方有分隔线（`border-t border-border`）
- [ ] 权限控制逻辑（`PermissionGuard`）保持不变
- [ ] 现有 Sidebar 测试全部通过

## Implementation Notes

将现有三个数组（`navItems`、`adminItems`、`teamItem`）重组为 `businessItems` 和 `adminItems` 两个数组，渲染逻辑相应简化。分隔线通过 `adminItems[0]` 的 className 中加 `mt-2 pt-2 border-t border-border` 实现。
