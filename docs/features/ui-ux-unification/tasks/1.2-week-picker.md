# 1.2: 新建 WeekPicker 共享组件

## Description

新建 `src/components/shared/WeekPicker.tsx`，实现跨浏览器兼容的周次选择器，支持上一周/下一周快速切换。同步新建组件测试文件。

## Reference Files

- `frontend/src/utils/weekUtils.ts`（依赖 1.1）
- `frontend/src/components/shared/WeekPicker.tsx`（新建）
- `frontend/src/components/shared/WeekPicker.test.tsx`（新建）
- `docs/features/ui-ux-unification/design/tech-design.md` — F3 节，WeekPicker Props 接口和实现骨架

## Acceptance Criteria

- [ ] 组件接受 `weekStart`、`onChange`、`maxWeek?`、`className?` 四个 props
- [ ] 显示格式为单行：`‹   2026年第16周  04/13 ~ 04/19   ›`
- [ ] 点击「‹」调用 `onChange(addWeeks(weekStart, -1))`
- [ ] 点击「›」调用 `onChange(addWeeks(weekStart, 1))`
- [ ] 当 `weekStart >= maxWeek` 时，「›」按钮 `disabled`，点击无响应
- [ ] 不使用 `<input type="week">`，Safari 15+ 可正常渲染
- [ ] 组件测试全部通过（5 个用例，覆盖率 ≥ 80%）

## Implementation Notes

- 使用 `lucide-react` 的 `ChevronLeft`、`ChevronRight` 图标
- 容器样式：`inline-flex items-center gap-1 h-8 rounded-md border border-border bg-white px-2`
- 禁用状态：`opacity-40 cursor-not-allowed`
- `maxWeek` 默认值在组件内计算：`getCurrentWeekStart()`
