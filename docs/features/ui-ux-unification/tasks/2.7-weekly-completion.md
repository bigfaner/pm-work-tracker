# 2.7: F7: 每周进展子事项显示完成度百分比

## Description

在 `WeeklyViewPage.tsx` 的 `SubItemRow` 组件中，子事项标题之后、负责人之前插入完成度百分比数字。

## Reference Files

- `frontend/src/pages/WeeklyViewPage.tsx`（`SubItemRow` 组件）
- `docs/features/ui-ux-unification/design/tech-design.md` — F7 节

## Acceptance Criteria

- [ ] 每个子事项行在标题文字之后显示完成度数字（如 `65%`）
- [ ] 完成度 < 100 时样式为 `text-[11px] font-semibold text-secondary`
- [ ] 完成度 = 100 时样式为 `text-[11px] font-semibold text-success-text`
- [ ] 始终显示，不论是否有进度变化
- [ ] 现有测试全部通过

## Implementation Notes

数据来源为 `SubItemSnapshot.completion`（已有字段），无需新增 API 调用。
