# 2.2: F2: 表格操作按钮加图标（4 个页面）

## Description

为用户管理、角色管理、团队详情、待办事项四个页面的表格行内操作按钮添加对应图标，统一为「图标 + 文字」风格。

## Reference Files

- `frontend/src/pages/UserManagementPage.tsx`
- `frontend/src/pages/RoleManagementPage.tsx`
- `frontend/src/pages/TeamDetailPage.tsx`
- `frontend/src/pages/ItemPoolPage.tsx`
- `docs/features/ui-ux-unification/design/tech-design.md` — F2 节，图标映射表

## Acceptance Criteria

- [ ] `UserManagementPage`：「编辑」加 `Pencil` 图标；「修改状态」根据用户状态动态显示 `ToggleRight`（启用）或 `ToggleLeft`（停用）
- [ ] `RoleManagementPage`：「编辑」加 `Pencil` 图标；「删除」加 `Trash2` 图标
- [ ] `TeamDetailPage`：「设为PM」加 `Crown` 图标；「移除」加 `UserMinus` 图标；「变更」加 `RefreshCw` 图标
- [ ] `ItemPoolPage`：「转为主事项」加 `ArrowUpCircle` 图标；「转为子事项」加 `ArrowDownCircle` 图标；「拒绝」加 `XCircle` 图标
- [ ] 图标规格：`w-3.5 h-3.5`，Button 内 flex 布局自动对齐
- [ ] 按钮 `variant`、`size`、`onClick`、`disabled` 逻辑完全不变
- [ ] 现有相关测试全部通过

## Implementation Notes

所有图标均来自已安装的 `lucide-react`，无需新增依赖。Button 组件已有 `gap-1.5` flex 布局，图标与文字自动对齐，无需额外样式。
