# 1.1: 新建 weekUtils.ts 工具函数

## Description

从 `WeeklyViewPage.tsx` 提取周次计算逻辑，新建 `src/utils/weekUtils.ts`，供 WeekPicker 组件和两个页面共用。

## Reference Files

- `frontend/src/pages/WeeklyViewPage.tsx`（提取来源）
- `frontend/src/utils/weekUtils.ts`（新建目标）
- `frontend/src/utils/weekUtils.test.ts`（新建测试）
- `docs/features/ui-ux-unification/design/tech-design.md` — F3 节，weekUtils.ts 导出接口

## Acceptance Criteria

- [ ] `src/utils/weekUtils.ts` 导出以下函数：`getCurrentWeekStart`、`getWeekNumber`、`getISOWeekYear`、`toLocalDateString`、`addWeeks`、`formatWeekLabel`
- [ ] `addWeeks` 使用 `setDate(date.getDate() + n * 7)` 实现，不使用毫秒计算（规避 DST 问题）
- [ ] `formatWeekLabel("2026-04-13")` 返回 `"2026年第16周  04/13 ~ 04/19"` 格式字符串
- [ ] `weekUtils.test.ts` 覆盖所有导出函数，包含跨年边界和 DST 边界测试用例
- [ ] `vitest --run` 全部通过，`weekUtils.ts` 覆盖率 100%

## Implementation Notes

- `getISOWeekYear`：处理跨年周次（如 12 月底的第 1 周属于下一年），使用 ISO 8601 标准
- `formatWeekLabel` 日期范围格式：`MM/DD ~ MM/DD`（月日各两位，补零）
- 提取后 `WeeklyViewPage.tsx` 中的本地函数暂不删除（等 2.3 任务替换时一并清理）
